# 🚀 Jorbites v2.3.0 ja és aquí!

Estem encantats d'anunciar el llançament de Jorbites v2.3.0! Aquesta actualització porta noves característiques, millores de rendiment i una millor experiència per a desenvolupadors.

## ✨ Què hi ha de nou?
- 🏅 **Insígnies**: S'ha afegit un nou endpoint d'insígnies (badges) per donar suport als assoliments dels usuaris.
- ⚡ **Rendiment**: Millora del rendiment general de l'estat i optimització de `RecipeModal` mitjançant l'ús de `useRef` per al seguiment de l'usuari actual.
- ♿ **Accessibilitat**: Millora dels elements de la interfície d'usuari mitjançant l'ús de components de botó adequats per a un millor suport interactiu.
- 🛠️ **Experiència del Desenvolupador (DX)**: Integració de React Doctor a GitHub Actions i refactorització de les funcions auxiliars d'errors d'API per a una major claredat.
- 🐛 **Correcció d'errors**: Solució de problemes en l'entrada de categories i resolució de tests inestables a `RelatedContentStep` causats per actualitzacions amb debounce.
- 📖 **Blogs de llançament**: S'han inclòs les publicacions oficials del blog per al llançament de la v2.2.5.
