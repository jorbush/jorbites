# 🚀 Jorbites v2.4.0 is out!

The latest version of Jorbites is now available! v2.4.0 brings new automated features, refactored components, and continued focus on application stability and performance.

## ✨ What's new

- 🤖 **Automated Features**: Implemented automated top recipe votation to streamline community highlights.
- 🏗️ **Architecture & Refactoring**: Abstracted the workshop modal for better reusability across the platform.
- ⚡ **Performance & Optimization**: Fixed `useRef` debounce implementation and resolved Next.js image performance warnings.
- 🧹 **Bug Fixes & Maintenance**: Fixed flaky tests in `textFormatting.test.tsx` caused by `window` being undefined and officially added the v2.3.9 release blog post.
