# 🚀 Jorbites v2.3.0 is here!

We're excited to announce the release of Jorbites v2.3.0! This update brings new features, performance enhancements, and improved developer experience.

## ✨ What's new
- 🏅 **Badges**: Added a new badges endpoint to support user achievements.
- ⚡ **Performance**: Improved overall state performance and optimized `RecipeModal` by using `useRef` for current user tracking.
- ♿ **Accessibility**: Enhanced UI elements by using proper button components for better interactive support.
- 🛠️ **Developer Experience**: Integrated React Doctor into GitHub Actions and refactored API error helper functions for better clarity.
- 🐛 **Bug Fixes**: Fixed category input issues and resolved flaky tests in `RelatedContentStep` caused by debounced updates.
- 📖 **Release Blogs**: Included the official blog posts for the v2.2.5 release.
