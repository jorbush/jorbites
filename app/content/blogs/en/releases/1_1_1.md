# 🚀 Jorbites v1.1.1 is here!

In this update, we've focused on polishing the recently introduced Lists feature. We've improved how lists look on smaller screens, refined the privacy management tools, and added more useful information to your lists.

## ✨ What's new
- ✨ **Lists UI Refinements**: Improved layout for small screens and responsive padding to ensure a consistent experience across all devices.
- ✨ **Enhanced List Management**: You can now see the creation date and owner of a list, and we've added the ability to delete lists.
- ✨ **Privacy Toggle Improvements**: Replaced text labels with intuitive padlock icons for privacy management, featuring optimized colors for both light and dark themes.
- ✨ **UI & UX Updates**: Updated the "Add Recipe List" button icon and added missing translations for loading states.
- ✨ **v1.1.0 Release Blog**: The official blog posts for the v1.1.0 release are now available.
