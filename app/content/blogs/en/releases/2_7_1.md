# 🚀 Jorbites v2.7.1 is here!

We're happy to share Jorbites v2.7.1, a maintenance release focused on polishing the user experience and improving our internal code structure.

## ✨ What's new

- 🎨 **UI Polish**: Refined our certificate components by adopting typographic ellipses for a more professional look.
- 🌍 **Better Localization**: Course durations are now fully translated, making it easier for everyone to understand the time commitment required for each course.
- ♿ **Contrast Fixes**: Resolved a contrast issue where gray text was used on colored backgrounds, ensuring better readability and accessibility.

## 🛠️ Technical Improvements

- 🧹 **State Management**: We've consolidated multiple `useState` calls into `useReducer` hooks in several components to simplify state logic and improve maintainability.
- ✅ **Test Coverage**: Added comprehensive unit tests for course duration translations to prevent future regressions.
