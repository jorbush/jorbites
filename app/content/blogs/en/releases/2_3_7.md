# 🚀 Jorbites v2.3.7 is out!

The latest version of Jorbites is now available! v2.3.7 brings a variety of performance optimizations, improves the developer experience by resolving various React warnings, and continues our commitment to codebase maintenance.

## ✨ What's new

- ⚡ **Performance & Optimization**: Implemented dynamic imports for the PDF library to reduce bundle size, replaced state with refs in performance-critical components, and hoisted pure functions to module scope to prevent unnecessary re-renders.
- 🛠️ **Developer Experience**: Resolved several React warnings (including exhaustive-deps and hook cleanup), abstracted `ALLOWED_LANGUAGES` into a centralized constants file, and refactored non-component exports to ensure Fast Refresh compatibility.
- 🧹 **Code Quality & Maintenance**: Fixed UI issues such as the three-dots ellipsis, removed unused dependencies like `jest-json-schema`, and updated test mocks to resolve accessibility (`no-noninteractive-element-interactions`) and event-handling (`no-prevent-default`) warnings.
- 📝 **Maintenance**: Officially added the v2.3.6 release blog post and removed the `isNew` indicator from the plannings section in the UserMenu.
