# 🚀 Jorbites v2.3.1 is out!

We're continuing to polish Jorbites with version 2.3.1! This update brings several accessibility fixes, performance enhancements using SWR, and resolutions for various framework warnings.

## ✨ What's new
- ⚡ **Performance**: Replaced several `useEffect` fetches with SWR for better caching and revalidation. Optimized immutable sorting using `toSorted()` and deferred pathname reads to reduce unnecessary re-renders.
- ♿ **Accessibility**: Fixed multiple accessibility warnings, including `anchor-has-content`, `heading-has-content`, and improvements in `QuestsClient`.
- 🛠️ **DX & Framework**: Resolved `server-after-nonblocking` and `async-defer-await` warnings to align with Next.js 15+ standards. Refactored `UserMenu` to use refs for click-outside logic.
- 📖 **Release Blogs**: Added the official blog posts for the v2.3.0 release.
