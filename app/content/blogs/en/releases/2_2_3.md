# 🚀 Jorbites v2.2.3 is here!

We're excited to announce the release of Jorbites v2.2.3! This update focuses on performance optimizations, accessibility fixes, and internal code improvements to make Jorbites even better.

## ✨ What's new
- ⚡ **Performance**: Parallelized independent awaits in server components to reduce request waterfalls and improve loading times.
- ♿ **Accessibility**: Fixed static element interaction warnings and replaced array indices with stable identifiers for better screen reader support and UI stability.
- 🔧 **Code Quality**: Resolved React destructuring warnings to ensure better compatibility with the React Compiler.
- 🧪 **Testing**: Upgraded Vitest from 3.0.9 to 4.1.0 for improved testing capabilities and stability.
- 📖 **Release Blogs**: Added the official blog post for the v2.2.2 release.
