# 🚀 Jorbites v2.8.0 is here!

We're excited to announce Jorbites v2.8.0, bringing more powerful tools to find and plan your meals, along with important maintenance updates.

## ✨ What's new

- 🔍 **Enhanced Filtering**: Finding the perfect recipe is now easier with new filters for calories, cuisine types, and yields.
- 📅 **Planning Improvements**: You can now see total calories directly within your meal plans, helping you stay on track with your nutritional goals.
- 🛠️ **UI Refinement**: We've relocated the date filter into the filter options for a cleaner and more intuitive search experience.
- ☀️ **Event Update**: The "Summer Sips" event date has been updated to July 11, 2026. Don't miss it!

## 🛠️ Technical Improvements

- 📊 **Architecture**: Updated our system diagrams to reflect the latest ecosystem changes.
- 🪵 **Logging**: Optimized Axiom logs for better performance and observability.
- ⚙️ **Code Health**: Refactored the large PATCH route handler in the recipe API and fixed a Next.js `useSearchParams` suspense warning to ensure a smoother, more reliable application.
