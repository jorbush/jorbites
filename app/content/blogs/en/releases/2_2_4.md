# 🚀 Jorbites v2.2.4 is here!

We're excited to announce the release of Jorbites v2.2.4! This update introduces security enhancements, UI fixes, and a new blog post about the connection between food and mood.

## ✨ What's new
- 🔒 **Security**: Added authentication checks to server actions to ensure data integrity and user privacy.
- 🎨 **UI/UX**: Fixed flexbox spacing warnings and resolved side effect warnings to improve interface stability and performance.
- 📝 **Content**: Published a new blog post: "Do you want to know the effects that what you eat has on your mood?", exploring how nutrition affects emotional well-being.
- 📖 **Release Blogs**: Included the official blog posts for the v2.2.3 release and improved their sorting logic.
