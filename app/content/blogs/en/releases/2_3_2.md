# 🚀 Jorbites v2.3.2 is out!

The latest version of Jorbites is now available! v2.3.2 focuses on improving performance through better data handling, enhancing SEO with missing metadata, and resolving several UI and notification bugs.

## ✨ What's new
- ⚡ **Performance**: Optimized O(n) lookups using Sets and resolved `js-combine-iterations` warnings by using `.reduce()`. Parallelized independent data fetching in server components to reduce request waterfalls.
- 🔍 **SEO**: Added missing metadata to SEO-critical pages to improve search engine visibility and social sharing.
- 🎨 **UI & UX**: Fixed the `no-pure-black-background` design warning and resolved `no-array-index-key` React warnings for better rendering stability. Removed unnecessary `useMemo` for simple expressions.
- 🛠️ **Bug Fixes**: Resolved a critical bug in push notifications related to `isInitialized`.
- 📖 **Release Blogs**: Added the official blog posts for the v2.3.1 release.
