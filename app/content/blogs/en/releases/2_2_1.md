# 🚀 Jorbites v2.2.1 is here!

We're rolling out Jorbites v2.2.1! This update focuses on refining our UI, updating upcoming event schedules, and continuing our commitment to code quality and accessibility.

## ✨ What's new
- 📅 **Event Updates**: Updated the dates for the "Pass the Bite" event, now scheduled for July 2026.
- 🎨 **UI Refinements**: Reduced heading font weights to semibold across the platform for better readability and a more modern look.
- 🧹 **Code Quality**: Optimized Tailwind CSS usage by adopting the `size-N` shorthand for better maintainability.
- 📖 **Release Blogs**: Added the official blog post for the v2.2.0 release.
