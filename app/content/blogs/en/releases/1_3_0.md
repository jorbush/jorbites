# 🚀 Jorbites v1.3.0 is here!

We're excited to bring you Jorbites v1.3.0! This update introduces the ability to generate recipe books, a fresh "liquid glass" UI style, improved support for iOS PWAs, and important stability fixes.

## ✨ What's new
- ✨ **Recipe Book**: Introduced automatic recipe book generation to easily compile your favorite recipes.
- ✨ **UI Design**: Implemented a new "liquid glass" styling for a more modern and fluid interface.
- ✨ **iOS PWA**: Improved the header experience specifically for iOS PWA users.
- ✨ **Stability**: Fixed module state issues in tracking actions to ensure more reliable data collection.
- ✨ **Dependencies**: Updated internal dependencies to maintain security and performance.
- ✨ **v1.2.3 Release Blog**: Added the official blog posts for the v1.2.3 release.
