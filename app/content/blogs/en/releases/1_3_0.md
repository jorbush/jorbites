# 🚀 Jorbites v1.3.0 is here!

We're excited to bring you Jorbites v1.3.0! This update introduces a beautiful new liquid glass design, improves the iOS PWA experience, and adds a new feature for generating recipe books.

## ✨ What's new
- ✨ **Design**: Added a new liquid glass styling for a more modern and polished look.
- ✨ **iOS PWA**: Improved the iOS PWA header for better integration and usability.
- ✨ **Recipe Book**: Added a new feature to generate recipe books.
- ✨ **Security & Performance**: Fixed mutable module state in tracking actions and updated dependencies.
- ✨ **v1.2.3 Release Blog**: Added the official blog posts for the v1.2.3 release.
