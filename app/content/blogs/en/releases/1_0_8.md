# 🚀 Jorbites v1.0.8 is here!

This version continues to improve the platform with new recipe options, UI bug fixes, and updated dependencies for better performance and security.

## ✨ What's new
- ✨ **New Preparation Method**: You can now select "Grilled" as a preparation method when creating or editing recipes.
- ✨ **New Recipe Category**: A new "Sauces" category has been added to better organize your favorite condiments.
- ✨ **UI Bug Fixes**: Resolved an issue where double tooltips would appear when hovering over badges in the Event Calendar.
- ✨ **Event Notification Refinements**: Improved the reliability of recurrent event notifications and refined "ending soon" alerts for better clarity.
- ✨ **Dependency Updates**: Updated core dependencies including `next`, `vite`, `axios`, and `lodash` to their latest versions to ensure stability and security.
- ✨ **v1.0.7 Release Blog**: Included the official blog post for the previous v1.0.7 release.
