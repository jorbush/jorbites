# 🚀 Jorbites v2.5.0 is out!

The latest version of Jorbites is now available! v2.5.0 focuses on SEO optimizations for our recipes, enhancing the voting experience, and keeping our platform up-to-date with architectural refinements.

## ✨ What's new

- 🔍 **SEO Enhancements**: Added new fields for recipes, including calories, cuisine type, and yield. These improvements help search engines better understand and display our delicious content.
- 🗳️ **Improved Voting UI**: Refined the voting interface to provide a smoother and more intuitive experience for the community.
- 🏗️ **Maintenance & Architecture**: Updated the architecture diagram and fixed React hook warnings in the workshop to ensure long-term maintainability.
- 📅 **Event Updates**: Rescheduled the Smash Burger Championship event and officially added the v2.4.1 release blog post.
