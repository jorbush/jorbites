# 🚀 Jorbites v1.1.2 is here!

In this update, we continue to refine the Lists feature with improved user interface elements and better management options. We've also addressed some technical debt and improved SEO by fixing canonical URLs.

## ✨ What's new
- ✨ **Enhanced List Management**: You can now delete recipes directly from your lists in the ListView, giving you more control over your collections.
- ✨ **UI & UX Polish**: Improved the AddRecipeToList modal interface and added missing cursor pointers to interactive elements like the padlock and trash can icons.
- ✨ **SEO & Technical Fixes**: Fixed canonical URLs to improve search engine optimization and ensured all release notes are up to date.
- ✨ **v1.1.1 Release Blog**: The official blog posts for the v1.1.1 release are now included.
