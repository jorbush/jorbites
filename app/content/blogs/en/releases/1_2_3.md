# 🚀 Jorbites v1.2.3 is here!

We're excited to bring you Jorbites v1.2.3! This update focuses on enhancing the weekly challenge experience with Server-Side Rendering (SSR), adding delicious new event content, and refining our UI and workflows.

## ✨ What's new
- ✨ **Weekly Challenge**: Added SSR support and fixed the events page integration for a smoother weekly challenge experience.
- ✨ **Events**: Introduced the new "Chocolate Lovers" event for all the cocoa enthusiasts.
- ✨ **UI Design**: Resolved design warnings in the FooterMenu to ensure a cleaner interface.
- ✨ **GitHub Workflows**: Optimized the execution timing for the weekly challenge GitHub workflow.
- ✨ **v1.2.2 Release Blog**: Added the official blog posts for the v1.2.2 release.
