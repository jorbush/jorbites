# 🚀 Jorbites v2.3.9 is out!

The latest version of Jorbites is now available! v2.3.9 continues our focus on refining the user experience with several performance optimizations, accessibility improvements, and key bug fixes to ensure a smoother application.

## ✨ What's new

- ⚡ **Performance & Optimization**: Optimized `useSearchParams` handling, improved image responsiveness with the `sizes` attribute, implemented `useEffectEvent` for better effect management, and added more Suspense boundaries for a better loading experience.
- ♿ **Accessibility**: Enhanced accessibility by removing `autoFocus` from search inputs, ensuring a more predictable focus flow for all users.
- 🛠️ **Code Quality & Refactoring**: Refactored state management to remove mirrored states and derived `useState`, hoisted static arrays to module scope, and modernized test utilities with `structuredClone`.
- 🧹 **Bug Fixes & Maintenance**: Fixed hydration errors, resolved missing event handlers in modals, and officially added the v2.3.8 release blog post.
