# 🚀 Jorbites v2.8.2 is here!

We are excited to announce Jorbites v2.8.2, featuring precise recipe filtering, optimized search state handling, and critical dependency updates.

## ✨ What's new

- 🍳 **Exact Cuisine Filtering**: Refined recipe search filtering to ensure exact matching for cuisine categories, preventing partial substring matches (e.g., no longer retrieving 'Latin American' recipes when filtering for 'American').
- 🔍 **Optimized Search Experience**: Resolved chain-state issues in the search interface to ensure highly responsive and efficient user inputs.

## 🛠️ Technical Improvements

- 📦 **Dependency Updates**: Upgraded `next` from 16.2.6 to 16.2.11, `next-auth` from 4.24.12 to 4.24.15, and `postcss` from 8.5.10 to 8.5.12 for enhanced stability, security, and rendering performance.
