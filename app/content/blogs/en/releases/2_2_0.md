# 🚀 Jorbites v2.2.0 is here!

We're excited to announce the release of Jorbites v2.2.0! This update brings new organization tools for your meal planning, including pinned recipes and per-meal limits. We've also improved SEO for planning pages and fixed some UI issues.

## ✨ What's new
- 📌 **Pinned Recipes**: You can now pin recipes to keep your favorites easily accessible.
- 🥗 **Meal Limits**: Added a limit to the number of recipes per meal in plannings to help you stay organized.
- 🔍 **SEO Improvements**: Added SEO metadata to Plannings pages for better visibility.
- 🛠️ **Fixes**: Fixed missing 'type' attribute on various button elements for better accessibility and standards compliance.
- 📖 **Release Blogs**: Added the official blog post for the v2.1.0 release.
