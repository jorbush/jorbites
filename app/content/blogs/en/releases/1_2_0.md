# 🚀 Jorbites v1.2.0 is here!

We're excited to bring you Jorbites v1.2.0! This update focuses on enhancing the user experience on mobile devices, improving our challenge system with new notifications, and strengthening our backend with standardized error handling and updated dependencies.

## ✨ What's new
- ✨ **Responsive List Headers**: Improved the ListView header to ensure a better experience on small screens, keeping your list actions accessible and clear.
- ✨ **Challenge Notifications**: Added a new notification system for weekly challenges to keep you updated on the latest culinary quests.
- ✨ **Standardized Error Handling**: Improved the consistency of our weekly-challenge API by implementing standardized error handling.
- ✨ **v1.1.3 Release Blog**: Added the official blog posts for the v1.1.3 release.
- ✨ **Dependency Update**: Updated `next` to v16.2.6 for improved stability and performance.
