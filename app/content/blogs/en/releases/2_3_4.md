# 🚀 Jorbites v2.3.4 is out!

The latest version of Jorbites is now available! v2.3.4 focuses on performance enhancements, UI refinements for events, and improved stability through better error handling and dependency updates.

## ✨ What's new
- ⚡ **Performance Optimizations**: Hoisted static I/O operations, implemented Suspense for better loading states, and added explicit caching strategies to server-side fetches.
- 📅 **Event Display Improvements**: Refined the display of recurrent events in event cards for better clarity.
- 🛠️ **Stability & DX**: Standardized error handling for list retrieval, updated core dependencies like Axios, and optimized components for the React Compiler.
- 📝 **Release Blogs**: Added the official blog posts for the v2.3.3 release.
