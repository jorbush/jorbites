# 🚀 Jorbites v2.3.6 is out!

The latest version of Jorbites is now available! v2.3.6 focuses on optimizing performance, refactoring state management, and improving overall code quality.

## ✨ What's new

- ⚡ **Performance Optimizations**: Optimized proxy images and parallelized asynchronous calls in routes for faster loading.
- 🛠️ **State Management**: Migrated from `useState` loadings to use transactions and implemented `useReducer` for better state handling.
- 🧹 **Code Quality**: Removed dead code, simplified ingredient and step input keys in the workshop, and adjusted log levels (using `warn` instead of `error` where appropriate).
- 📝 **Bug Fixes & Docs**: Fixed a `BlogDetailClient` runtime error and added the official v2.3.5 release blog posts.
