# 🚀 Jorbites v2.0.0 is here!

We're thrilled to announce the milestone release of Jorbites v2.0.0! This version brings significant new features including recipe book configuration and a new planning system, along with several improvements and fixes.

## ✨ What's new
- ✨ **Features**: Implemented the new planning system and added a recipe book configuration modal.
- ✨ **Fixes**: Resolved an issue with the "next" steps in the recipe navigation.
- ✨ **Release Blogs**: Added the official blog posts for the v1.3.3 release.
