# 🚀 Jorbites v1.2.2 is here!

We're excited to bring you Jorbites v1.2.2! This update focuses on improving our development workflow by migrating to pnpm, enhancing our SEO with the Metadata API, and maintaining a high standard of accessibility and security.

## ✨ What's new
- ✨ **Package Management**: Migrated the project from npm to pnpm for faster and more efficient dependency management.
- ✨ **Metadata API**: Migrated policy pages to the Next.js Metadata API for better SEO and performance.
- ✨ **Accessibility**: Fixed missing alt attributes in test mocks to improve testing standards.
- ✨ **v1.2.1 Release Blog**: Added the official blog posts for the v1.2.1 release.
- ✨ **Dependency Updates**: Updated `postcss` and `uuid` to their latest versions for security and stability.
