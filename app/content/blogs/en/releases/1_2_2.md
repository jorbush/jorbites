# 🚀 Jorbites v1.2.2 is here!

We're excited to announce the release of Jorbites v1.2.2! This update brings important infrastructure changes with the migration to pnpm, improves our SEO and accessibility, and keeps our dependencies up to date.

## ✨ What's new
- ✨ **Migration to pnpm**: Switched our package manager to pnpm for faster, more efficient builds and better dependency management.
- ✨ **SEO & Metadata**: Migrated policy pages to the Next.js Metadata API for improved search engine optimization.
- ✨ **Accessibility**: Fixed missing alt attributes in test mocks to ensure better accessibility standards in our testing environment.
- ✨ **v1.2.1 Release Blog**: Added the official blog posts for the v1.2.1 release.
- ✨ **Dependencies**: Updated `postcss` and `uuid` to their latest versions for improved security and stability.
