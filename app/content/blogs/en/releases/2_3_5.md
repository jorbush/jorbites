# 🚀 Jorbites v2.3.5 is out!

The latest version of Jorbites is now available! v2.3.5 brings several accessibility refinements, fixes for hydration issues, and further code optimizations.

## ✨ What's new
- ♿ **Accessibility Improvements**: Enhanced accessibility for icons, dots indicators, and the comment button.
- 🔧 **Hydration & Component Fixes**: Resolved hydration errors related to translations, comments, and workshops. Fixed issues with component mounting.
- 🛠️ **Code Quality & Security**: Extracted default prop values to module-level constants and removed `dangerouslySetInnerHTML` from `RecipeSchema` for better security.
- 📈 **Performance**: Addressed several Page Insights issues to improve overall site performance.
- 📝 **Release Blogs**: Added the official blog posts for the v2.3.4 release.
