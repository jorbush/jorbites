# 🚀 Jorbites v2.2.5 is here!

We're excited to announce the release of Jorbites v2.2.5! This update introduces several performance optimizations, code refactoring for better maintainability, and important bug fixes.

## ✨ What's new
- ⚡ **Performance**: Optimized Quests filter state with lazy initialization, memoized `Intl.DateTimeFormat` in `WorkshopHead`, and improved concurrency in the Cloudinary utility.
- 🛠️ **Refactoring**: Renamed event handlers across the codebase to be more descriptive and follow project standards.
- 🐛 **Bug Fixes**: Resolved the `no-pass-live-state-to-parent` warning in the `TranslateableContent` component to ensure better state synchronization.
- 📖 **Release Blogs**: Included the official blog posts for the v2.2.4 release.
