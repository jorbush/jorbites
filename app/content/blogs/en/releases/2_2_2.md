# 🚀 Jorbites v2.2.2 is here!

We're excited to announce the release of Jorbites v2.2.2! This update focuses on improving accessibility, fixing hydration issues, and streamlining our dependencies.

## ✨ What's new
- ♿ **Accessibility**: Improved accessibility by removing redundant ARIA roles and ensuring all interactive controls have proper labels.
- 🔧 **Hydration Fix**: Resolved hydration rendering issues in the recipe client button for a smoother user experience.
- 🧹 **Dependency Cleanup**: Removed the unused `lodash.debounce` dependency to keep our bundle lean.
- 📖 **Release Blogs**: Added the official blog post for the v2.2.1 release.
