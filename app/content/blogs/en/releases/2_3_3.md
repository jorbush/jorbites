# 🚀 Jorbites v2.3.3 is out!

The latest version of Jorbites is now available! v2.3.3 includes internal refactoring to improve component maintainability, accessibility fixes in the comments section, and the official release blogs for v2.3.2.

## ✨ What's new
- 🏗️ **Refactoring**: Abstracted large components into smaller, more manageable sub-components to improve code readability and maintainability.
- ♿ **Accessibility**: Fixed an accessibility warning in the Comment component to ensure a more inclusive experience for all users.
- 📖 **Release Blogs**: Added the official blog posts for the v2.3.2 release.
