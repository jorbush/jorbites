# 🚀 Jorbites v2.4.1 is out!

The latest version of Jorbites is now available! v2.4.1 continues to refine the platform with architectural abstractions, performance gains, and UI polish.

## ✨ What's new

- 🏗️ **Architecture & Refactoring**: Abstracted translatable logic to improve internationalization and code maintainability.
- ⚡ **Performance & Optimization**: Optimized recipe voting by fixing sequential independent awaits and using `array.toSorted()` for more efficient immutable sorting.
- 🎨 **UI & Styling**: Refined the visual design by using `font-semibold` for headings and modernized Tailwind CSS usage with the `size-N` shorthand.
- 🧹 **Bug Fixes & Maintenance**: Officially added the v2.4.0 release blog post.
