# 🚀 Jorbites v2.5.1 is out!

The latest version of Jorbites is now available! v2.5.1 focuses on enhancing the recipe book generation experience, improving performance, and refining the user interface.

## ✨ What's new

- 📚 **Recipe Book Enhancements**: Added the ability to select specific recipes when generating a book and fixed recipe images for the selection process.
- ⚡ **Performance Boost**: Improved the performance of the events page for a smoother browsing experience.
- 🎨 **UI Improvements**: Fixed the top recipe badge card for better visual consistency.
- 🏗️ **Architecture Updates**: Added Tagatoni to the architecture diagram.
- 📝 **Documentation**: Officially added the v2.5.0 release blog post.
