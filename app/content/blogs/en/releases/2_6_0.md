# 🚀 Jorbites v2.6.0 is here!

We're excited to announce the release of Jorbites v2.6.0! This update brings new features like the Likes page, important fixes for video uploads, and under-the-hood performance improvements.

## ✨ What's new

- ❤️ **Likes Page**: A dedicated page to view all your liked recipes has been added, with improved idempotency for a better user experience.
- 📹 **Video Field Fix**: Resolved issues with the uploaded video field to ensure smooth media handling.
- 📅 **Event Updates**: The Burger Championship and Cocktail Contest have been rescheduled to August 2026.
- 🌐 **Internationalization**: Recipe categories are now translated during the recipe book selection step.
- ⚡ **Performance & Optimization**: Parallelized independent async operations in API routes for faster responses and adopted the `size-N` Tailwind shorthand for cleaner code.
- 🎨 **UI Refinements**: Fixed bold heading issues in the LikesClient component.
