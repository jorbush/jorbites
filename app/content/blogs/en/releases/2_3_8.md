# 🚀 Jorbites v2.3.8 is out!

The latest version of Jorbites is now available! v2.3.8 brings a variety of performance enhancements, improves accessibility, and continues our commitment to a cleaner and more efficient codebase through strategic refactoring.

## ✨ What's new

- ⚡ **Performance & Optimization**: Implemented passive event listeners in the PullToRefresh component and optimized utility files by replacing repeated array lookups with more efficient Map/Set operations.
- ♿ **Accessibility**: Added descriptive accessible labels to various interactive controls to ensure a better experience for users relying on assistive technologies.
- 🛠️ **Code Quality & Refactoring**: Refactored the Modal component to eliminate derived state, transitioned complex rendering functions into dedicated components, and addressed React warnings related to `no-prop-callback-in-effect`.
- 🧹 **Bug Fixes & Maintenance**: Resolved Suspense-related issues, fixed layout bugs in the Navbar, and officially added the v2.3.7 release blog post.
