# 🚀 ¡Jorbites v2.7.1 ya está aquí!

¡Nos complace compartir Jorbites v2.7.1, una versión de mantenimiento enfocada en pulir la experiencia del usuario y mejorar nuestra estructura de código interna!

## ✨ ¿Qué hay de nuevo?

- 🎨 **Pulido de la interfaz**: Refinados nuestros componentes de certificados adoptando elipsis tipográficas para un aspecto más profesional.
- 🌍 **Mejor localización**: Las duraciones de los cursos ahora están totalmente traducidas, lo que facilita que todos entiendan el tiempo necesario para cada curso.
- ♿ **Correcciones de contraste**: Resuelto un problema de contraste donde se usaba texto gris sobre fondos de colores, asegurando una mejor legibilidad y accesibilidad.

## 🛠️ Mejoras técnicas

- 🧹 **Gestión de estado**: Hemos consolidado múltiples llamadas a `useState` en ganchos (hooks) `useReducer` en varios componentes para simplificar la lógica de estado y mejorar el mantenimiento.
- ✅ **Cobertura de pruebas**: Añadidas pruebas unitarias exhaustivas para las traducciones de duración de los cursos para evitar futuras regresiones.
