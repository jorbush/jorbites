# 🚀 ¡Jorbites v2.3.7 ya está disponible!

¡La última versión de Jorbites ya está disponible! La v2.3.7 trae una variedad de optimizaciones de rendimiento, mejora la experiencia del desarrollador al resolver varios avisos de React y continúa nuestro compromiso con el mantenimiento del código.

## ✨ ¿Qué hay de nuevo?

- ⚡ **Rendimiento y optimización**: Implementación de importaciones dinámicas para la librería PDF para reducir el tamaño del bundle, reemplazo de estados por refs en componentes críticos para el rendimiento y elevación de funciones puras al ámbito del módulo para evitar renderizados innecesarios.
- 🛠️ **Experiencia del desarrollador**: Resolución de varios avisos de React (incluyendo exhaustive-deps y limpieza de hooks), abstracción de `ALLOWED_LANGUAGES` en un archivo de constantes centralizado y refactorización de exportaciones que no son componentes para asegurar la compatibilidad con Fast Refresh.
- 🧹 **Calidad del código y mantenimiento**: Corrección de problemas de UI como los puntos suspensivos, eliminación de dependencias no utilizadas como `jest-json-schema` y actualización de mocks de prueba para resolver avisos de accesibilidad (`no-noninteractive-element-interactions`) y manejo de eventos (`no-prevent-default`).
- 📝 **Mantenimiento**: Se ha añadido oficialmente la publicación del blog del lanzamiento v2.3.6 y se ha eliminado el indicador `isNew` de la sección de planificaciones en el Menú de Usuario.
