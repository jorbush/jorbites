# 🚀 ¡Jorbites v2.3.0 ya está aquí!

¡Estamos encantados de anunciar el lanzamiento de Jorbites v2.3.0! Esta actualización trae nuevas características, mejoras de rendimiento y una mejor experiencia para desarrolladores.

## ✨ ¿Qué hay de nuevo?
- 🏅 **Insignias**: Se ha añadido un nuevo endpoint de insignias (badges) para dar soporte a los logros de los usuarios.
- ⚡ **Rendimiento**: Mejora del rendimiento general del estado y optimización de `RecipeModal` mediante el uso de `useRef` para el seguimiento del usuario actual.
- ♿ **Accesibilidad**: Mejora de los elementos de la interfaz de usuario mediante el uso de componentes de botón adecuados para un mejor soporte interactivo.
- 🛠️ **Experiencia del Desarrollador (DX)**: Integración de React Doctor en GitHub Actions y refactorización de las funciones auxiliares de errores de API para una mayor claridad.
- 🐛 **Corrección de errores**: Solución de problemas en la entrada de categorías y resolución de tests inestables en `RelatedContentStep` causados por actualizaciones con debounce.
- 📖 **Blogs de lanzamiento**: Se han incluido las publicaciones oficiales del blog para el lanzamiento de la v2.2.5.
