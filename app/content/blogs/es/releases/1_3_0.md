# 🚀 ¡Jorbites v1.3.0 ya está aquí!

¡Estamos encantados de presentarte Jorbites v1.3.0! Esta actualización introduce la capacidad de generar libros de recetas, un nuevo estilo de interfaz "liquid glass", soporte mejorado para las PWA de iOS y correcciones de estabilidad importantes.

## ✨ ¿Qué hay de nuevo?
- ✨ **Libro de Recetas**: Se introdujo la generación automática de libros de recetas para recopilar fácilmente tus recetas favoritas.
- ✨ **Diseño de Interfaz**: Se implementó un nuevo estilo "liquid glass" para una interfaz más moderna y fluida.
- ✨ **PWA en iOS**: Se mejoró la experiencia del encabezado específicamente para los usuarios de PWA en iOS.
- ✨ **Estabilidad**: Se corrigieron problemas de estado del módulo en las acciones de seguimiento para garantizar una recopilación de datos más confiable.
- ✨ **Dependencias**: Se actualizaron las dependencias internas para mantener la seguridad y el rendimiento.
- ✨ **Blog de lanzamiento v1.2.3**: Se añadieron las publicaciones oficiales del blog para el lanzamiento de la v1.2.3.
