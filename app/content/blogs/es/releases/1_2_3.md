# 🚀 ¡Jorbites v1.2.3 ya está aquí!

¡Estamos encantados de presentarte Jorbites v1.2.3! Esta actualización se centra en mejorar la experiencia del desafío semanal con Renderizado del Lado del Servidor (SSR), añadir contenido delicioso para eventos y refinar nuestra interfaz y flujos de trabajo.

## ✨ ¿Qué hay de nuevo?
- ✨ **Desafío Semanal**: Se añadió soporte para SSR y se corrigió la integración de la página de eventos para una experiencia de desafío semanal más fluida.
- ✨ **Eventos**: Se introdujo el nuevo evento "Amantes del Chocolate" para todos los entusiastas del cacao.
- ✨ **Diseño de Interfaz**: Se resolvieron advertencias de diseño en el FooterMenu para garantizar una interfaz más limpia.
- ✨ **Flujos de Trabajo de GitHub**: Se optimizó el tiempo de ejecución para el flujo de trabajo de GitHub del desafío semanal.
- ✨ **Blog de lanzamiento v1.2.2**: Se añadieron las publicaciones oficiales del blog para el lanzamiento de la v1.2.2.
