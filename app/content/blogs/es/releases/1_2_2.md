# 🚀 ¡Jorbites v1.2.2 ya está aquí!

¡Estamos encantados de presentarte Jorbites v1.2.2! Esta actualización se centra en mejorar nuestro flujo de trabajo de desarrollo mediante la migración a pnpm, mejorar nuestro SEO con la API de Metadatos y mantener un alto estándar de accesibilidad y seguridad.

## ✨ ¿Qué hay de nuevo?
- ✨ **Gestión de Paquetes**: Se migró el proyecto de npm a pnpm para una gestión de dependencias más rápida y eficiente.
- ✨ **API de Metadatos**: Se migraron las páginas de políticas a la API de Metadatos de Next.js para un mejor SEO y rendimiento.
- ✨ **Accesibilidad**: Se corrigieron los atributos alt faltantes en los mocks de las pruebas para mejorar los estándares de prueba.
- ✨ **Blog de lanzamiento v1.2.1**: Se añadieron las publicaciones oficiales del blog para el lanzamiento de la v1.2.1.
- ✨ **Actualizaciones de Dependencias**: Se actualizaron `postcss` y `uuid` a sus últimas versiones para mejorar la seguridad y estabilidad.
