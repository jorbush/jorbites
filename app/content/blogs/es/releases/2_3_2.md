# 🚀 ¡Jorbites v2.3.2 ya está disponible!

¡La última versión de Jorbites ya está disponible! La v2.3.2 se centra en mejorar el rendimiento mediante un mejor manejo de datos, mejorar el SEO con metadatos faltantes y resolver varios errores de interfaz de usuario y notificaciones.

## ✨ ¿Qué hay de nuevo?
- ⚡ **Rendimiento**: Se han optimizado las búsquedas O(n) mediante el uso de Sets y se han resuelto las advertencias `js-combine-iterations` utilizando `.reduce()`. Se ha paralelizado la obtención de datos independientes en componentes de servidor para reducir las cascadas de peticiones.
- 🔍 **SEO**: Se han añadido metadatos faltantes a páginas críticas para el SEO para mejorar la visibilidad en los motores de búsqueda y el uso compartido en redes sociales.
- 🎨 **UI y UX**: Se ha corregido la advertencia de diseño `no-pure-black-background` y se han resuelto las advertencias de React `no-array-index-key` para una mejor estabilidad de renderizado. Se ha eliminado `useMemo` innecesario para expresiones simples.
- 🛠️ **Correcciones de errores**: Se ha resuelto un error crítico en las notificaciones push relacionado con `isInitialized`.
- 📖 **Blogs de lanzamiento**: Se han añadido las publicaciones oficiales del blog para el lanzamiento de la v2.3.1.
