# 🚀 ¡Jorbites v2.0.0 ya está aquí!

¡Estamos encantados de anunciar el lanzamiento del hito Jorbites v2.0.0! Esta versión trae nuevas funciones significativas, incluyendo la configuración del libro de recetas y un nuevo sistema de planificación, junto con varias mejoras y correcciones.

## ✨ ¿Qué hay de nuevo?
- ✨ **Funciones**: Se ha implementado el nuevo sistema de planificación y se ha añadido un modal de configuración del libro de recetas.
- ✨ **Correcciones**: Se ha resuelto un problema con los pasos "siguientes" en la navegación de recetas.
- ✨ **Blogs de lanzamiento**: Se han añadido las publicaciones oficiales del blog para el lanzamiento de la v1.3.3.
