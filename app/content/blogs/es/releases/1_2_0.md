# 🚀 ¡Jorbites v1.2.0 ya está aquí!

¡Estamos encantados de presentarte Jorbites v1.2.0! Esta actualización se centra en mejorar la experiencia del usuario en dispositivos móviles, mejorar nuestro sistema de retos con nuevas notificaciones y fortalecer nuestro backend con un manejo de errores estandarizado y dependencias actualizadas.

## ✨ ¿Qué hay de nuevo?
- ✨ **Cabeceras de Lista Adaptables**: Se ha mejorado la cabecera de la Vista de Lista (ListView) para garantizar una mejor experiencia en pantallas pequeñas, manteniendo las acciones de tu lista accesibles y claras.
- ✨ **Notificaciones de Retos**: Se ha añadido un nuevo sistema de notificaciones para los retos semanales para mantenerte al día con las últimas misiones culinarias.
- ✨ **Manejo de Errores Estandarizado**: Se ha mejorado la consistencia de nuestra API de retos semanales implementando un manejo de errores estandarizado.
- ✨ **Blog de lanzamiento v1.1.3**: Se han añadido las publicaciones oficiales del blog para el lanzamiento de la v1.1.3.
- ✨ **Actualización de Dependencias**: Se ha actualizado `next` a la v16.2.6 para mejorar la estabilidad y el rendimiento.
