# 🚀 ¡Jorbites v2.6.0 ya está aquí!

¡Estamos encantados de anunciar el lanzamiento de Jorbites v2.6.0! Esta actualización trae nuevas características como la página de "Me gusta", correcciones importantes para la subida de vídeos y mejoras de rendimiento internas.

## ✨ ¿Qué hay de nuevo?

- ❤️ **Página de "Me gusta"**: Se ha añadido una página dedicada para ver todas tus recetas favoritas, con una mejor idempotencia para una mejor experiencia de usuario.
- 📹 **Corrección del campo de vídeo**: Se han resuelto problemas con el campo de vídeo subido para asegurar un manejo de medios fluido.
- 📅 **Actualizaciones de eventos**: El Campeonato de Hamburguesas y el Concurso de Cócteles se han reprogramado para agosto de 2026.
- 🌐 **Internacionalización**: Las categorías de recetas ahora se traducen durante el paso de selección del libro de recetas.
- ⚡ **Rendimiento y Optimización**: Se han paralelizado operaciones asíncronas independientes en las rutas de la API para respuestas más rápidas y se ha adoptado el atajo `size-N` de Tailwind para un código más limpio.
- 🎨 **Refinamientos de la interfaz de usuario**: Se han corregido problemas de encabezados en negrita en el componente LikesClient.
